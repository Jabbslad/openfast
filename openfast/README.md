# OpenFast

A privacy-first fasting and wellness tracker that runs entirely in your browser.

## Features

- **Fasting Timer** — Start, pause, and end fasting windows with real-time elapsed tracking
- **Meal Log** — Record meals with timestamps and optional notes
- **Hydration Tracking** — Log water and liquid intake toward daily goals
- **Progress Dashboard** — View streaks, badges, and historical fasting data
- **Streak & Badge System** — Earn badges for consistency and milestone achievements
- **Settings & Tips** — Customise fasting goals and browse curated wellness guides
- **Offline-First PWA** — Installable, works without a network connection
- **100% Local Storage** — All data stays on your device; nothing is sent to any server
- **Export / Import / Clear** — Full control over your data at any time

## Quick Start

```bash
git clone https://github.com/openfast/openfast.git
cd openfast
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

React 18 + TypeScript + Vite + Tailwind CSS + Dexie (IndexedDB) + vite-plugin-pwa
